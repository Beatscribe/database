Freebie-GBs-2019.gb
===================

Heebie-GBs made a return in 2019, and this version has some major technical
improvements. The LSDj GUI is no longer shown, and instead custom graphics,
animated to the rhythm of the music can be shown. The ROM is also shrunk to
512 kB, which was achieved by removing parts of LSDj that were unnecessary for
the function of the modded player. 

This free ROM does not include any of the easter eggs, which are exclusive to
the cartridge release.
